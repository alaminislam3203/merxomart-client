## Project Overview

**MerxoMart** is a responsive and interactive **e-commerce web application**
designed for online shoppers to explore, buy, and manage products efficiently.
Users can browse products, add them to their cart, manage orders, and admins can
manage products in a dashboard.

This project leverages **Next.js (App Router)**, **Firebase Authentication**,
and modern frontend libraries for animations, state management, and
notifications.

---

## Features

### User Authentication

- Sign up, log in, and log out functionality.
- Firebase Authentication integration.
- Protected routes for authenticated users.

### Product Management

- Browse all products with images, descriptions, and pricing.
- Product detail page for each product.
- Admins can add, edit, or delete products.

### Cart & Orders

- Add products to cart, update quantity, and remove items.
- Order summary with tax and total calculations.
- Select shipping address and apply promo codes.
- Place orders (mock functionality; backend integration with MongoDB).

### Interactive UI

- SweetAlert2 notifications for actions like delete, order placed, etc.
- Responsive design for mobile, tablet, and desktop.
- Interactive buttons and hover effects.

### Navbar & Footer

- Responsive navbar with login/logout and cart status.
- Footer includes company info, contact, and social links.
- Social links: Facebook, Twitter, Instagram, LinkedIn.

## Technologies & NPM Packages Used

- **Next.js (v16)**: React framework with App Router
- **React**: Component-based UI library
- **Tailwind CSS**: Utility-first styling
- **Firebase**: Authentication
- **MongoDB**: Backend database
- **React Icons**: Icons library
- **SweetAlert2**: Beautiful alerts & confirmations
- **Framer Motion**: Optional animations
